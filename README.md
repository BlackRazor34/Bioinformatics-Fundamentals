## Overview
This project simulates the biological process of the Central Dogma using Python and Biopython. It performs:
1. **DNA Analysis:** Calculates GC content and stability.
2. **Transcription:** Converts DNA sequences into mRNA.
3. **Translation:** Translates mRNA into protein sequences suitable for drug discovery simulations.

## Tech Stack
* Python 3.13.9
* Biopython Library
* VSCode Notebook
